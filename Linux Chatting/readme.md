# 1. Overview
* 리눅스 소켓프로그래밍을 통한 실시간 단체 채팅 및 파일 교환 프로그램
* #quit키워드를 통한 개별 로그아웃
* #exit키워드를 통한 전체 접속 종료
* [File]키워드를 통한 클라이언트간 P2P통신
